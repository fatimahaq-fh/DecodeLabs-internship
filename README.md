# DecodeLabs UI/UX Design Internship

##  About
This repository contains all the projects and assignments completed during my **UI/UX Design Internship at DecodeLabs**. Throughout this internship, I learned the complete UI/UX design process, including wireframing, high-fidelity UI design, design systems, and interactive prototyping using Figma.

##  Internship Objectives
- Learn UI/UX Design Fundamentals
- Create Low-Fidelity Wireframes
- Design High-Fidelity User Interfaces
- Build a Professional Design System
- Create Interactive Prototypes
- Improve User Experience and Visual Design Skills
