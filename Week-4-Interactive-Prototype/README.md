# Week 4 – Interactive Prototype

## Project
FreshNest – Grocery Delivery App

## Overview
This project was developed as part of the DecodeLabs UI/UX Design Internship. The focus of Week 4 was to transform the static high-fidelity design into an interactive prototype using Figma.

## Prototype Features
- Splash Screen Auto Transition
- Homepage Navigation
- Product List Navigation
- Product Details Screen
- Add to Cart Interaction
- Success Popup Overlay
- Continue Shopping Flow
- Smart Animate Transitions
- Interactive Navigation
