# Week 3 – High-Fidelity UI Design
## Organic Food Delivery App
This project was created as part of the **DecodeLabs UI/UX Design Internship**. The goal was to transform low-fidelity wireframes into a modern, user-friendly high-fidelity interface using Figma.

## Deliverables
- 🎨 Style Tile (Design System)
- 🚀 Splash Screen
- 🏠 Homepage
- 📋 Product List Screen
- 🥑 Product Details Screen
  
## Features
- Modern Organic Food Theme
- Professional Color Palette
- Typography System
- Reusable Components
- Responsive Desktop Layout
- Consistent Spacing & Alignment
- Clean and User-Friendly Interface
